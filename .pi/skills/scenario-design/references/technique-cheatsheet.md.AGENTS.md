# scenario-design/references/technique-cheatsheet.md — index

How-to per design/resilience technique with repo examples. EP: split input classes (invalid too). BVA: six values min-1..max+1. Decision table: flag combos, mark impossible rows. State-transition: legal + illegal edges. State-convergence: assert invariants not "visible after N ms". Performance: workload + metric + threshold + window; no threshold ⇒ gap. Fault injection: delay + abort per dependency. Maps scenario classes → techniques. Rejects Gherkin/BDD.
