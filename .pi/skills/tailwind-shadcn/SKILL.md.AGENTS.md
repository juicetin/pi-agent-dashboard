# tailwind-shadcn/SKILL.md — index

Tailwind + shadcn/ui patterns. Install via `npx shadcn@latest init` / `add <component>`. CSS-variable theming (`--background`, `--primary`, `--radius`, `[data-theme="earth"]`). shadcn component usage: `Button` variants (default/secondary/outline/ghost/destructive, sizes sm/lg), `Card`/`CardHeader`/`CardContent`, `Accordion`, `Form`/`FormField`/`FormItem`. `cn()` helper = `twMerge(clsx(...))`. Responsive mobile-first prefixes, dark mode via `dark:` prefix. Animation utilities (transition-all, animate-pulse/spin/bounce).
