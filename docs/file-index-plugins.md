| `packages/jj-plugin/package.json` | **NEW** workspace package (change: add-jj-workspace-plugin). Carries `pi-dashboard-plugin` manifest claiming `session-card-badge` (`JjWorkspaceBadge`, predicate `isInJjWorkspace`), `session-card-action-bar` (`JjActionBar`, predicate `isInJjRepo`), `sidebar-folder-section` (`JjWorkspaceList`), `command-route /jj` (`JjWorkspaceView`), `settings-section` general tab (`JjPluginSettings`). | `packages/jj-plugin/src/client/JjActionBar.tsx` | Session-card row of buttons (Workspace / Fold back / Forget). Per Decision 10. |
