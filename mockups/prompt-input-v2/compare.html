Mobile adaptation rules

  • Persistent row = ＋ · model (flex-fills) · ⋯ · send. Nothing else stays inline.
  • Send / stop ≥ 44px (WCAG 2.5.5), always thumb-reachable at the row's right edge.
  • ＋ = bottom-sheet: attach (photo · file · screenshot) + tools (Explore · Preview).
  • ⋯ = overflow: thinking level · Steer | Queue delivery · inline terminal.
  • Context strip collapses to chips-only, no group labels.
  • Footer condenses to one line and drops the newline hint; context budget stays.
