Improve frontmatter rendering

Why

The markdown renderer loads no frontmatter plugin, so YAML headers are misparsed into a giant heading. Every .md surface that carries frontmatter is affected.

What changes

  • Add remark-frontmatter + a YAML parser.
  • Render an Obsidian-style Properties panel, collapsed by default.
  • Add a frontmatter prop to MarkdownContent so chat opts out.
