SessionCard redesign — v2

Actions stay where they live today; only the design changes. The old WORKSPACE subcard is split into GIT (branch / PR / worktree / Push / Open PR / Merge / Close) and JJ (jj pill / Add workspace / Fold back / Forget) so the two VCSs stop mixing. Other additions: pills+lines stepper inside OPENSPEC, colour-coded buttons, animated multicolour neon border on the selected card, and a cloned action strip directly above the prompt textarea so you don't lose context while typing.
