# OpenSpecActivityBadge.tsx — index

Session-card sub-badge showing active OpenSpec phase. Exports `OpenSpecActivityBadge`. `phaseLabels`/`phaseColors` map every `OpenSpecPhase` (explore, new, continue, ff, apply, verify, archive, sync-specs, onboard) to label + color. Renders phase, optional `changeName`, optional `completedTasks/totalTasks` progress.
