# ServerSelector.tsx — index

Server selector dropdown showing persisted known servers. Availability probing runs **only on dropdown open** (once per open, not on mount, not on timer, not while closed). Accepts `inFlightSwitchKey` to render spinner on entry being switched to. Unreachable entries `disabled`, rendered with `opacity-50` + `cursor-not-allowed`, clicks no-ops. Reachable clicks delegate to transactional staging-socket switch (see `server-switch.ts`). See change: fix-remote-connect-cors-gates — adds exported `isLanHost(host)` (RFC-1918/CGNAT/link-local/`.local`); a probe `.catch()` (opaque transport failure) on a LAN host renders "CORS-blocked — allowlist this origin on the remote" (amber, disabled, `data-cors-blocked`) instead of "Unreachable" (browser can't tell CORS-block from connection-refused; LAN literal is the trusted-network proxy).
