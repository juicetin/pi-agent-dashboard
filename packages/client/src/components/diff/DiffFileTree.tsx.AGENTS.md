# DiffFileTree.tsx — index

Two-level file tree of changed files. Exports `DiffFileTree`, `FileSelection`. Builds tree via `buildFileTree` (`lib/diff-tree`). Dir nodes expand/collapse; file nodes show modified(●)/added(+) indicator, expand to per-change events when `changes.length > 1`. Selects via `onSelect({ filePath, changeIndex })`. File rows show per-file `+adds −dels` counts; aggregate header shows `totalAdditions`/`totalDeletions` with `summed` badge. See change: add-change-summary-table. Tool/mixed rows render an `OriginBadge` (`data-testid=origin-badge`) + `created by <producedBy>` label (title = full command). Accepts `otherChanges?` prop → muted, collapsed `▸ N other working-tree changes` group (`data-testid=other-changes-group`, file rows hidden until expanded) + a header `this session only` toggle (`data-testid=session-only-toggle`) that hides the group. See change: detect-tool-created-files.
