# chat-virtual-rows.ts — index

Pure helpers for the windowed (TanStack Virtual) transcript. Exports `isBurst`/`isGroup` guards, `virtualRowKey(item,index)` (CR-3: `burst.id` \| first-member id \| `group-<index>` \| `msg.id`), `estimateVirtualRowSize(item, textChars?)` (CONTENT-AWARE pre-measure: `base + ceil(textChars/80)*20` clamped 8000px + per-kind image reserve 300 user/512 toolResult; burst 220/group 64 constants), `computeRowTextChars(item)` (aggregate rendered text length, walked once in ChatView useMemo), `buildTurnToFirstRowIndex(rows)` (CR-4: `turnIndex`→first display-row index over the FILTERED rows, backs `scrollToTurn`), `rangeToRowIndexSpan(range, container, rowCount)` (DOM Range → inclusive `{min,max}` display-row span; walks endpoints to nearest `[data-index]`, clamps non-virtual endpoints via `compareDocumentPosition`, normalizes reversed/same-row, null when no virtual row touched), `extendRangeWithSelection(base, span, cap, count)` (union selection span into `defaultRangeExtractor` output; returns `base` unchanged past `cap` so caller actively clears — no full mount). See change: preserve-chat-selection-during-churn. See change: virtualize-chat-transcript-tanstack. See change: fix-chat-scroll-to-top-estimate-drift — static per-role estimate under-shot large top rows 10-50x so scroll-up never converged; content-aware estimate shrinks the delta that drives TanStack's built-in above-viewport correction.
