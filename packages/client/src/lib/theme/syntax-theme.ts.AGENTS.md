# syntax-theme.ts — index

Single source of truth for prism syntax styles in client. `getSyntaxTheme(resolved, themeName)` resolves prism palette for active app theme + runs every result through pure helper `stripTokenBackgrounds(style)` which removes `background` / `backgroundColor` from every selector containing `.token` (including `.token.deleted` / `.token.inserted` diff washes inside fenced ```diff blocks). Wrapper selectors (`pre[class*="language-"]`, `code[class*="language-"]`) left untouched so panel background stays overridable to `var(--bg-code)`. Fallback path (unknown theme) also runs through strip. Pinned by `packages/client/src/lib/__tests__/syntax-theme.test.ts` (13 assertions across base/dracula/nord/github/catppuccin × dark/light). See change: strip-token-backgrounds-in-code-blocks.
