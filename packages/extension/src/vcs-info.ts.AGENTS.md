# vcs-info.ts — index

Gathers git branch/remote/PR/worktree info via shared platform git helpers. Exports `GitInfo`, `detectBranch`, `detectRemoteUrl`, `detectPrNumber`, `detectWorktree`, `detectIsGitRepo`, `gatherGitInfo`. `detectWorktree` compares `--git-common-dir` vs `--show-toplevel`. `detectIsGitRepo` returns tri-state `boolean\|undefined` from `git.isGitRepo()` Result: ok→value, exit-128→false, else→undefined (never false on inconclusive probe); attached to `session_register` (session-sync.ts, bridge.ts). Builds links via `buildGitLinks`. Returns undefined if not a repo. Exports `gatherGitStatus(cwd)` → `GitStatus\|undefined` via shared `git.gitStatusV2` (`git status --porcelain=v2 --branch`). See changes: gate-session-worktree-button-on-git, add-session-uncommitted-indicator-and-commit.
