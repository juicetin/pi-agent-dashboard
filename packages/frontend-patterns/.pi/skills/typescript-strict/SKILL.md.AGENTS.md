TypeScript strict patterns. Config: `strict`, `noUncheckedIndexedAccess`, `exactOptionalPropertyTypes`. `interface` for extendable objects, `type` for unions/tuples/primitives. Type guards (`value is Service`), discriminated unions (`Result<T>`). Generics: `fetchData<T>`, `ListProps<T>`, polymorphic `BoxProps<T extends React.ElementType>`. Utility types (Partial/Required/Pick/Omit/Record/Extract/Exclude). Nullable handling: optional chaining + `??`, `== null` narrowing. `as const` for immutable arrays/objects. Zod schema type inference via `z.infer`.
