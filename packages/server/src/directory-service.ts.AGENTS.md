# directory-service.ts — index

Server-side directory-scoped ops. Exports `createDirectoryService`, `DirectoryService` interface, `isOpenSpecDataEmpty`, `hasOpenSpecDir`, `hasOpenSpecRoot`, `fnv1a32`, `phaseOffsetMs`, re-exports `effectiveMtimeOr`, `DiscoveredSession`. Session discovery + worker-pooled event hydration (`cancelLoad`), OpenSpec polling (mtime-gated cache, `fs.watch` change watcher, worker pool with pre-serialized payload, jitter, semaphore, TOCTOU gate), pi-resources scan (5× cadence), folder-HEAD poll+watcher, `reconfigurePolling`, `onDirectoryAdded`. Master `enabled:false` gate short-circuits to cleared state. Per-turn event-loop attribution: `scheduleOpenSpecTick` self-records `{at,ms,turn}` into `options.eventLoopSpikes` when a single synchronous turn's `performance.now()` run ≥ `eventLoopSpikeFloorMs` (default 100) — `tickOpen` (setInterval fire prefix), `dirPollPre` (setTimeout fire incl. `pollOne` sync prefix before the worker await), `dirPollPost` (broadcast continuation after the await); never summed across the await. Per-turn slow-tick alarm at `perTurnWarnMs` (default 250) names the turn, independent of the retained wall `durationMs > TICK_SLOW_WARN_MS` alarm. See change: attribute-openspec-poll-eventloop-stalls.
