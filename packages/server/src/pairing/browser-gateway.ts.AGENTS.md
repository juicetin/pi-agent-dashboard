# browser-gateway.ts — index

WebSocket gateway for browser clients. Exports `BrowserGateway` interface, `createBrowserGateway`, `buildOpenSpecConnectSnapshot`. Routes `BrowserToServerMessage` switch to `browser-handlers/*`. Manages subscriptions, replay suppression, headless `headlessPidRegistry`, `pendingResumeRegistry`, `viewedSessionTracker`, pending UI/prompt request replay, `broadcastEvent`/`broadcastSession*`/`broadcastOpenSpecUpdate` (pre-serialized payload fanout), plugin `registerHandler`/`registerDisconnectHandler`. Back-pressure guard via `MAX_WS_BUFFER`. Exports `getDroppedFrameStats()` (per-hop server→browser drop counters); `recordDroppedFrame` instruments each `MAX_WS_BUFFER` drop, surfaced on `/api/health#droppedFrames`. See change: add-custom-roles-ui. Routes `subagent_resync_request` → `handleSubagentResyncRequest` → `piGateway.sendToSession`. See change: fix-subagent-live-detail-reliability (D2).
