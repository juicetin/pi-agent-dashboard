# pi-resource-scanner.ts — index

Discovers extensions, skills, prompts, agents from local `.pi/`, global `~/.pi/agent/`, and installed `settings.json packages[]`. Agents from `agents/*.md` → `type:"agent"`; `parseFrontmatter` also reads `model` + `tools` (list `[a,b]`→`a,b` or scalar). Missing `agents/` → empty. Resolves package entries; local wins on dedup. Stamps `PiResource.enabled` from `pi-resource-activation.resolveActivation` (agents always true — pi has no agent activation dim; injectable via `ScanOptions.resolveActivation`; default true). Exports `scanPiResources`, `scanLocalResources`, `scanGlobalResources`, `parseFrontmatter`, `ScanOptions`, types from shared. `_resetNpmRootCache` test helper. See change: folder-resource-activation-toggle, resources-card-tabs.
