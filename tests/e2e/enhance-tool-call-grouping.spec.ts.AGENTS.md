# enhance-tool-call-grouping.spec.ts — index

Playwright spec for universal tool-call grouping (change: enhance-tool-call-grouping). 3 tests, faux model. `[[faux:grp-single]]` → a SINGLE bash call forms a `tool-burst-group` whose collapsed `tool-burst-summary` shows the tool's own one-liner (`echo single-call`), NOT `1 tool calls`. `[[faux:burst-heterogeneous]]` → multi header `3 tool calls` + `tool-burst-breakdown` chips. `[[faux:grp-reasoning]]` (3 bash + trailing `thinking`+text; PATCHes a COMPLETE displayPrefs w/ reasoning:true + all toolCalls:true — a sparse patch on an unseeded container would hide all tools) → expanding the group body shows the absorbed reasoning as a real `reasoning-block` (ThinkingBlock, labeled Reasoning), zero `tool-burst-narration`. Run w/ `PW_CHANNEL=chrome` for system browser. Needs `PI_E2E_SEED=1`.
